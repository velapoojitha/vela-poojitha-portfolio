Guess The Number

  • Tech Stack: HTML, CSS, Javascript
  • Live URL: Live Link
  • Github URL: Project Link

  • The Guess the Number Game is an interactive browser-based application where users attempt to guess a randomly generated number between 1 and 100. The app provides instant feedback on each guess, tracks the number of attempts, and allows users to restart the game seamlessly. Built using HTML, CSS, and JavaScript, this project focuses on core programming logic, conditional statements, and dynamic DOM manipulation. It strengthened my understanding of event handling, real-time user interaction, and responsive UI feedback design. This project showcases my ability to implement logical problem-solving concepts into engaging and user-friendly web applications.
