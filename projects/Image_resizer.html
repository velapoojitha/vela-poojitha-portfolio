Image-Resizer

  • Tech Stack: HTML, CSS, Javascript
  • Live URL: Live Link
  • Github URL: Project Link

  • The Image Resizer Web App is a lightweight and user-friendly tool that allows users to upload images and resize them dynamically based on custom width and height inputs. The application provides instant preview functionality and ensures optimized image scaling while maintaining visual quality. Built using HTML, CSS, and JavaScript, this project focuses on DOM manipulation, file handling, and real-time image processing in the browser. It strengthened my understanding of working with image objects, canvas-based resizing techniques, and responsive UI design. This project demonstrates my ability to build practical frontend utilities that combine functionality, performance, and clean user experience.
